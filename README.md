## <u>Teamate</u> Project Background
- ### 필요성
  - 카메라가 다양한 분야에서 활용되고 있지만, 렌즈 오염으로 인한 영상 품질 저하 문제가 지속적으로 발생하고 있다. 특히 AI 기반의 영상 시스템에서는 데이터 분석 오류나 성능 저하가 발생할 수 있다.
  - 축산업 분야의 스마트팜에서는 환경 특성상 먼지, 습기, 거미줄, 파리 배설물 등으로 인해 카메라 렌즈가 쉽게 오염되며, 이로 인해 객체 인식 및 데이터 분석의 신뢰도가 낮아지는 문제가 있다. 이러한 문제가 해결되지 않으면 자동화된 축산 관리 시스템이 오작동할 수 있으며, 결과적으로 생산성 및 운영 효율이 감소할 수 있다.
  - 자율주행 분야에서는 주행 중 카메라를 이용하여 주변 환경을 인식, 분석하는데, 주행 중 빗물, 흙, 벌레 등의 오염물질이 카메라 렌즈에 부착되면 차선 인식, 신호등 인식 등의 객체 인식 정확도를 떨어뜨릴 수 있다. 이는 자율주행 시스템의 오작동을 초래하며, 교통사고로 이어질 수 있다.
- ### 기존 해결책의 문제점
  - 기존의 단순 블러 임계값 등 단일 품질지표만으로는 반투명한 물방울, 거미줄처럼 얇은 구조물, 점 형태 이물질 등 서로 다른 오염을 일관되게 판별하기 어렵다.
  - 기존의 연구 및 실험은 사람이 물을 뿌리거나 분진을 인위적으로 발생시키는 방식이 많아, 실제 축산 현장의 다양한 오염 패턴을 반영하지 못하는 문제점이 있다.
  
## System Design
  - ### System Requirements
    - 기능 요구사항
      1) 다양한 오염 유형 인지 및 분류: 오염 유형별 패턴/색상/밀도/분포 분석
      2) 오염도 수치화 : 카메라 프레임 내 오염 면적 비율을 반영해 오염도를 정량지표로 산출
      3) 검출 성능 평가 : 육안 판독과 비교해 신뢰도 확인
      4) 환경 변화 : 조명/온도/습도 등 다양한 조건에서의 오염 감지 확인
    - 비기능 요구사항
      1) 엣지 디바이스(라즈베리파이 등)에서도 운용 가능(경량화)
      2) 오염 유형 추가 및 갱신에 용이한 데이터셋 제작
      3) 자동 청소장치 및 관제 시스템과의 연동성 강화
    
## Case Study
  - ### Description
    - 환경 
        - 돈사 CCTV 및 Soiling 공개 데이터셋
    - 블록 기반 특징/분류 성과
        - 채도 평균: 오염 영역 채도 경향성 확인
        - 색상 다양도(hue_std): 오염 영역 단조화 확인
        - 라플라시안 분산: 오염 영역 고주파 정보 감소 확인

  
## Conclusion
  - ### 현장 기반 오염 감지 표준화
      - 자연 발생 오염에 기반한 데이터셋으로 효과적인 오염 감지 성능 확보
  - ### 운영 연계 가능성
      - 자동 청소장치 와의 연동을 전제로, 오염이 고착화하기 전 청소시스템 가동으로 장비 수명 및 데이터 신뢰도 유지 기여
